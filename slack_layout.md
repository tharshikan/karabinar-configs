## ðŸš€ Workflows & Examples

Here is how to chain these keys for common tasks (assuming you're holding **Nav Layer 4**):

### 1. Navigation Flow
*   **Browsing History:**
    *   Went down a rabbit hole? Tap **O** to go **Back**.
    *   Went too far? Tap **I** to go **Forward**.
*   **Checking New Stuff:**
    *   Tap **P** to see **All Unreads**.
    *   Use **Arrows** to scroll through the list.
    *   Tap **Esc** (standard) or **R** to mark a read channel as complete.

### 2. Replying to a Thread
1.  Use **Down Arrow** (on Layer 4) to highlight the message you want to reply to.
2.  Tap **T** to open the **Thread** pane.
3.  Release Layer 4 and type your reply.

### 3. Searching / Jumping
*   Want to go to the `#general` channel?
    *   Tap **E** (opens Quick Switcher).
    *   Release Layer 4, type `gen`, hit Enter.

### 4. Adding Code
*   **Inline Code:** Type backticks around text: `` `code` ``.
*   **Code Block:** Type triple backticks:
    ```
    ```
    (your code)
    ```
*   **Shortcut:** Highlight code, then press `Cmd+Shift+C` (Standard macOS shortcut, not on Layer 4).

### 5. Editing Flow
1.  Make a typo? Tap **U** (sends Up Arrow).
2.  Your last message is now editable.
3.  Fix the typo.
4.  Hit **Enter** to save changes.

### 6. Managing Conversations
*   **Switching Contexts:**
    *   Need to talk to a person? Tap **M** (DMs).
    *   Need to find a channel? Tap **E** (Search).
*   **Saving for Later:**
    *   Find a message you need to act on later.
    *   Hover/Select it with cursors.
    *   Tap **B** (Saved) to bookmark it.
    *   Later, tap **B** (Saved) again to view your list.

### 7. Triage Mode
1.  Tap **P** (Unreads) to see everything new.
2.  Use **Arrows** to go down the list.
3.  Read a channel.
4.  Tap **R** (Read/Esc) to mark it done and go back to the list.
5.  Repeat until inbox zero!

### 8. Formatting Shortcuts (Standard Mac)
*   **Bold:** `Cmd+B`
*   **Italic:** `Cmd+I`
*   **Link:** `Cmd+Shift+U`
